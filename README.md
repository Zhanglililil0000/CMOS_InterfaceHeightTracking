# CMOS 高度控制系统

基于 CMOS 相机和 SIGMA KOKI 电动位移台的闭环高度控制系统。通过分析指示光在 CMOS 上的成像位置，PID 控制器驱动垂直位移台实时补偿高度偏移，实现样品高度的精确稳定。

## 工作原理

```
┌──────────┐    指示光     ┌──────────┐
│  样品     │ ──────────→ │  CMOS相机  │
└────┬─────┘   反射偏移    └────┬─────┘
     │                         │
     │  高度变化 → 光斑左右移动   │  图像采集（~25 FPS）
     │                         ▼
     │                  ┌──────────────┐
     │                  │  光斑检测     │
     │                  │  (质心法)     │
     │                  └──────┬───────┘
     │                         │  spot_x（当前 X 坐标）
     │                         ▼
     │                  ┌──────────────┐
     │                  │  滑动平均滤波  │
     │                  │  (默认10帧)   │
     │                  └──────┬───────┘
     │                         │  smoothed_x（平滑后的 X）
     │                         ▼
     │                  ┌──────────────┐
     └─────────────────│ PID 控制器    │
      位移台上下移动     │  error→位移量 │
                       └──────────────┘
```

1. 指示光经样品反射进入 CMOS 相机，光斑的 **X 坐标** 与样品高度线性相关（CMOS X 轴像素更多，作为高度探测方向）
2. 每帧光斑 X 坐标送入**滑动平均滤波器**，用最近 N 帧（默认 10 帧，约 0.4 秒）的均值替代单帧位置，消除液体晃动噪声
3. 用户将样品调至期望高度，点击 **标定** — 系统将当前平滑 X 值填入目标输入框
4. PID 控制器根据平滑后的 X 值与目标值的误差，驱动位移台补偿高度偏移

### 方向约定

| 操作 | 物理方向 | 高度 |
|------|----------|------|
| 光斑右移（X 增大） | 高度升高 | ↑ |
| 光斑左移（X 减小） | 高度降低 | ↓ |
| ▲ 步进+ | 电机上升 | ↑ |
| ▼ 步进- | 电机下降 | ↓ |

## 操作指南

### 连接设备

1. **连接相机**：点击右侧面板「枚举相机」→ 选择设备 → 「打开相机」，左侧窗口开始显示实时图像
2. **连接位移台**：输入 COM 端口号（默认 COM5）→ 点击「连接位移台」

### 标定与稳高

1. **标定**：手动将样品调至期望高度 → 点击「标定当前位置」，系统记录当前**平滑 X 均值**并自动填入目标 X 输入框
2. **稳高**：点击「开始稳高」→ 系统 PID 自动调节位移台保持光斑位置不变 → 点击「停止稳高」结束
3. **手动设目标**：也可以在目标 X 输入框中直接输入期望的 X 坐标值，再点击「开始稳高」即可追踪指定像素位置

### 手动控制

- **步进移动**：输入步进距离（脉冲数）→ 点击 `步进+` / `步进-` 移动指定距离
- **停止**：点击「停止」按钮立即停止位移台运动
- **搜零**：点击「搜零」执行位移台回零操作

### PID 参数说明

在「稳高控制」面板中可实时调整，修改后需重新点击「开始稳高」生效：

| 参数 | 说明 | 默认值 | 建议范围 |
|------|------|--------|----------|
| Kp | 比例系数，控制校正力度 | 0.3 | 0.1 ~ 0.5 |
| 死区 | 平滑误差小于此值（像素）时停止调整 | 2.0 | 1.0 ~ 5.0 |
| 平滑窗口 | 滑动平均的帧数 | 10 | 1 ~ 50 |

### 平滑窗口工作原理

采用**滑动窗口平均**（rolling window），每一帧都产出新的平滑值：

```
帧1:  窗口=[v1]              平滑值=(v1)/1
帧2:  窗口=[v1, v2]          平滑值=(v1+v2)/2
帧10: 窗口=[v1..v10]         平滑值=sum/10
帧11: 窗口=[v2..v11]         平滑值=sum/10  (v1 被挤出)
帧12: 窗口=[v3..v12]         平滑值=sum/10
```

- 窗口 = 1 → 无平滑，使用原始光斑位置
- 窗口 = 10 → 最近 10 帧均值（约 0.4 秒窗口）
- 窗口 = 30 → 约 1.2 秒窗口，适合晃动较大的液面

### PID 参数调试建议

| 现象 | 调整方向 |
|------|----------|
| 电机来回振荡 | ↑ 死区 / ↓ Kp / ↑ 平滑窗口 |
| 响应太慢、修正不及时 | ↓ 死区 / ↑ Kp |
| 稳定液体样品 | 死区 2.0~3.0, Kp 0.2~0.3, 窗口 10~20 |
| 晃动较大的液面 | 死区 3.0~5.0, Kp 0.1~0.2, 窗口 20~50 |

### 状态面板

| 指标 | 说明 |
|------|------|
| 光斑 X (高度) | 当前检测到的光斑 X 质心坐标（子像素精度） |
| 平滑 X | 滑动窗口平均后的 X 坐标（PID 使用的输入） |
| 光斑 Y | 光斑 Y 质心坐标（垂直方向） |
| 误差 | 目标 X - 平滑 X |
| 误差条 | 绿色（小）→ 黄色（中）→ 红色（大） |
| 位移台位置 | 当前位置（软件累积推算） |
| 帧率 | 相机实时采集帧率 |

### 修改 PID 高级参数

可在 [height_controller.py](src/height_controller.py) 中修改未暴露到 UI 的参数：

```python
self.Kp = 0.3           # 比例系数
self.Ki = 0.0           # 积分系数
self.Kd = 0.0           # 微分系数
self._max_step = 500    # 单次最大脉冲移动量
self._dead_zone = 2.0   # 死区（像素）
self._min_interval = 0.3 # 最小校正间隔（秒）
```

### 修改光斑检测参数

编辑 `src/server.py` 中 SpotDetector 初始化行：

```python
spot_detector = SpotDetector(
    window_size=30,       # 质心计算窗口大小（像素）
    threshold_ratio=0.2   # ROI 内阈值比例（相对于最大值）
)
```

## 常见问题

**Q: 启动后提示 SDK initialize failed？**
A: 确保已安装海康威视 MVS 客户端，系统环境变量 `MVCAM_COMMON_RUNENV` 已正确设置。

**Q: 位移台连接失败？**
A: 检查 COM 端口号是否正确（Windows 设备管理器 → 端口 查看），确认没有其他程序占用该端口。

**Q: 光斑检测不准确？**
A: 调整 `window_size` 和 `threshold_ratio` 参数，或确保指示光在相机视野内形成清晰亮斑。

**Q: 稳高不响应或震荡？**
A: 参考上方的「PID 参数调试建议」表格调整参数。稳定液体样品可增大死区（2.0~3.0）、减小 Kp（0.2~0.3）；晃动样品需增大平滑窗口（30~50）。

**Q: 目标 X 显示为空？**
A: 先点击「标定当前位置」，系统会自动填入当前平滑 X 值。也可以手动在输入框中输入期望值。

**Q: 步进距离与实际不符？**
A: 位移台内部有 1:2 的单位换算（1 用户脉冲 = 2 控制器脉冲），程序已自动处理换算，显示的步进距离和位置均为用户单位。
